# 期貨商保證金一覽表

每天自動更新的「期貨商原始保證金一覽表」靜態網站。手機閱讀優先，最上方有字級調整與「可用保證金」即時換算可下單口數的功能。

資料來源：臺灣期貨交易所（TAIFEX）。國外交易所。

---

## 檔案說明

| 檔案 | 用途 |
|---|---|
| `build.py` | 抓資料 + 產生 `index.html`。純標準函式庫。 |
| `.github/workflows/update.yml` | 自動排程設定。 |
| `index.html` | 產生的網站（執行後才會出現）。 |
| `data.json` | 除錯用的資料備份（執行後才會出現）。 |

## 自己在電腦上測試（選用）

```bash
python build.py     # 產生 index.html，用瀏覽器打開即可預覽
```
